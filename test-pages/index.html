OpenBridge Test Pages

Static HTML pages for integration testing of browser extension tools.

  • Form Testing
    Text inputs, email, password, textarea, select, checkbox, radio buttons, and form submission
  • Navigation Testing
    Anchor links, new tab links, programmatic navigation, and hash changes
  • Dynamic DOM Testing
    Adding/removing elements, toggling visibility, changing content, and dynamic forms
  • Keyboard Event Testing
    Key press display, keyboard shortcuts, contenteditable, and event logging
  • File Upload Testing
    Single and multiple file inputs, drag-and-drop zone, and file info display
